# 🎮 Monster Block Game - 게임 가이드

## 🎯 게임 시작하기

### 1. 게임 실행
- GameApp을 실행하면 시작 화면이 표시됩니다
- **START 버튼**을 클릭하여 게임을 시작하세요

### 2. 화면 구성
게임 화면은 3개의 주요 영역으로 구성됩니다:

#### 좌측 상단 - 점수 패널
- **점수**: 현재 획득한 점수
- **생명**: 남은 생명 개수 (최대 5개)
- **콤보**: 3콤보 이상일 때 표시 (주황색)

#### 좌측 하단 - 프로필 패널
- **부기 캐릭터**: 게임 상태를 나타내는 캐릭터
  - 😊 웃는 얼굴: 생명 4개 이상
  - 😐 평범한 얼굴: 생명 2-3개
  - 😢 슬픈 얼굴: 생명 1개 이하
- **큰 색상 박스**: 현재 맞춰야 할 색상
- **작은 색상 박스**: 다음에 맞춰야 할 색상

#### 우측 - 게임 플레이 영역
- **타겟 몬스터**: 3줄 6열로 배치된 색상 몬스터와 벽돌
- **슈터**: 화면 하단의 빨간색 사각형 (플레이어)
- **총알**: 슈터에서 발사되는 빨간색 원

## 🎮 조작 방법

### 기본 조작
- **A 키**: 슈터를 왼쪽으로 이동
- **D 키**: 슈터를 오른쪽으로 이동
- **Enter 키**: 총알 발사
- **P 키**: 게임 일시정지 / 재개
- **R 키**: 게임 재시작

### 조작 팁
1. A와 D 키를 꾹 누르면 슈터가 계속 이동합니다
2. 키에서 손을 떼면 슈터가 멈춥니다
3. Enter 키를 연타하여 여러 발의 총알을 발사할 수 있습니다

## 🎯 게임 규칙 상세

### 점수 시스템
| 상황 | 점수 변화 | 생명 변화 | 콤보 변화 |
|------|----------|---------|---------|
| 올바른 색상 맞춤 (기본) | +100점 | 변화 없음 | +1 |
| 올바른 색상 맞춤 (3콤보) | +150점 | 변화 없음 | +1 |
| 올바른 색상 맞춤 (5콤보) | +200점 | 변화 없음 | +1 |
| 올바른 색상 맞춤 (10콤보) | +300점 | 변화 없음 | +1 |
| 잘못된 색상 맞춤 | -50점 | -1 | 0 (리셋) |
| 벽돌 맞춤 | 변화 없음 | 변화 없음 | 유지 |

### 콤보 시스템 🔥
연속으로 올바른 색상을 맞추면 콤보가 증가하여 더 많은 점수를 획득할 수 있습니다:
- **1-2콤보**: 100점 (기본 점수)
- **3-4콤보**: 150점 (1.5배)
- **5-9콤보**: 200점 (2배)
- **10콤보 이상**: 300점 (3배)

**주의**: 잘못된 색상을 맞추면 콤보가 0으로 리셋됩니다!

### 하이스코어 시스템 🏆
- 최고 기록이 자동으로 저장됩니다
- 신기록 달성 시 게임 오버 화면에 "★ 신기록! ★" 표시
- 이전 최고 기록과 비교하여 자신의 실력 향상 확인 가능

### 색상 종류
게임에는 3가지 색상이 있습니다:
- 🔴 **빨간색** (RED)
- 🟢 **초록색** (GREEN)
- 🟡 **노란색** (YELLOW)

### 타겟 종류
- **빨간 몬스터**: 빨간색 타겟
- **초록 몬스터**: 초록색 타겟
- **노란 몬스터**: 노란색 타겟
- **벽돌**: 색상 변경 아이템 (어떤 색상이든 맞춰도 안전)

## 🎲 게임 전략

### 초급 전략
1. **색상 확인이 최우선**
   - 좌측 패널의 큰 색상 박스를 항상 확인하세요
   - 해당 색상의 몬스터만 조준하세요

2. **신중한 발사**
   - 잘못 쏘면 생명이 줄어듭니다
   - 확실할 때만 발사하세요

3. **벽돌 활용**
   - 원하는 색상이 없을 때 벽돌을 맞춰 색상을 변경하세요
   - 벽돌은 안전하게 맞출 수 있습니다 (생명 감소 없음)

### 중급 전략
1. **다음 색상 미리 파악**
   - 작은 색상 박스로 다음 색상을 미리 확인하세요
   - 현재 색상을 맞추면서 다음 색상의 위치를 파악하세요

2. **효율적인 이동**
   - 왼쪽부터 오른쪽으로 순서대로 맞추기
   - 또는 가장 많은 타겟이 모여있는 곳부터 공략

3. **일시정지 활용**
   - 복잡한 상황에서는 P키로 일시정지
   - 전략을 세운 후 다시 시작

### 고급 전략
1. **콤보 유지 전략**
   - 콤보를 유지하면 점수가 기하급수적으로 증가합니다
   - 10콤보 달성 시 한 타겟당 300점!
   - 확실한 타겟만 조준하여 콤보 유지

2. **생명 관리**
   - 생명이 2개 이하일 때는 더욱 신중하게
   - 불확실한 상황에서는 벽돌을 우선 공략
   - 부기의 표정으로 현재 상태 파악

3. **패턴 파악**
   - 타겟의 배치 패턴 파악
   - 자주 나오는 색상 조합 기억
   - 다음 색상을 미리 확인하여 전략 수립

## 🎯 게임 상태

### 진행 중
- 자유롭게 슈터를 조작하고 총알을 발사할 수 있습니다
- P키로 언제든지 일시정지 가능

### 일시정지
- 화면에 "일시정지" 문구가 표시됩니다
- 모든 게임 조작이 불가능합니다
- P키를 다시 눌러 게임 재개

### 게임 오버
- 생명이 0이 되면 게임 오버
- 최종 점수와 최고 기록이 화면에 표시됩니다
- 신기록 달성 시 "★ 신기록! ★" 메시지 표시
- R키를 눌러 새 게임 시작

## 💡 자주 묻는 질문 (FAQ)

### Q: 벽돌을 맞추면 어떻게 되나요?
A: 벽돌을 맞추면 점수나 생명에 변화 없이 다음 색상으로 변경됩니다. 안전하게 색상을 바꿀 수 있는 아이템입니다.

### Q: 모든 타겟을 깨면 어떻게 되나요?
A: 자동으로 새로운 3줄의 타겟이 생성됩니다. 게임은 계속 진행됩니다.

### Q: 생명은 어떻게 회복하나요?
A: 현재 버전에서는 생명 회복 기능이 없습니다. 5개의 생명으로 최대한 높은 점수를 노리세요!

### Q: 총알을 한 번에 여러 발 쏠 수 있나요?
A: 네, Enter 키를 여러 번 누르면 여러 발의 총알을 동시에 발사할 수 있습니다.

### Q: 게임을 중간에 저장할 수 있나요?
A: 현재 버전에서는 저장 기능이 없습니다. 한 판을 끝까지 플레이하세요.

### Q: 콤보는 어떻게 쌓이나요?
A: 연속으로 올바른 색상을 맞추면 콤보가 증가합니다. 3콤보부터 보너스 점수가 적용되며, 10콤보 이상이면 한 타겟당 300점을 획득할 수 있습니다!

### Q: 하이스코어는 어디에 저장되나요?
A: 게임 폴더의 highscore.txt 파일에 자동으로 저장됩니다. 파일을 삭제하면 하이스코어가 초기화됩니다.

### Q: 부기 캐릭터의 표정은 왜 바뀌나요?
A: 부기는 현재 생명 상태를 나타냅니다. 생명이 많으면 웃고, 적으면 슬퍼합니다. 부기의 표정을 보고 현재 상태를 빠르게 파악하세요!

## 🏆 고득점을 위한 팁

1. **콤보가 핵심**: 10콤보를 유지하면 기본 점수의 3배! 확실한 타겟만 조준하여 콤보 유지
2. **실수 최소화**: 잘못된 색상을 맞추면 점수 -50점, 생명 -1, 콤보 리셋이므로 신중하게 조준
3. **전략적 벽돌 활용**: 원하는 색상이 많이 남아있을 때 벽돌로 색상 변경
4. **전체 파악**: 타겟 전체를 보고 가장 효율적인 순서 계획
5. **부기 표정 주시**: 생명이 적어지면 더욱 신중하게 플레이
6. **침착함 유지**: 서두르다 실수하지 말고 침착하게 플레이
