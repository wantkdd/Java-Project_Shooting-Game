# 🎮 Monster Block Game (몬스터 블록 게임)

### 제작 : 유상완

![image](https://github.com/user-attachments/assets/0e39782a-3a4c-49bb-a0e3-8297aaa18180)

## 📝 게임 소개

Monster Block Game은 벽돌깨기와 슈팅 게임이 결합된 Java Swing 기반의 아케이드 게임입니다.
플레이어는 슈터를 좌우로 조작하여 색상이 맞는 몬스터를 맞추고 점수를 획득합니다.

## 🎯 게임 룰

### 기본 규칙
- **목표**: 현재 지정된 색상과 일치하는 몬스터를 맞춰 점수를 획득
- **생명**: 시작 시 5개의 생명 보유
- **점수 시스템**:
  - ✅ 올바른 색상의 몬스터 맞춤: **+100점**, 다음 색상으로 변경
  - ❌ 잘못된 색상의 몬스터 맞춤: **-50점**, **생명 -1**
  - 🧱 벽돌(Brick) 맞춤: 점수 변화 없음, 다음 색상으로 변경

### 게임 진행
1. 좌측 패널에서 **현재 맞춰야 할 색상**(큰 박스)과 **다음 색상**(작은 박스)을 확인
2. 슈터를 좌우로 이동하여 올바른 색상의 몬스터를 조준
3. 총알을 발사하여 타겟을 맞춤
4. 모든 타겟을 깨면 새로운 타겟이 자동 생성
5. 생명이 0이 되면 게임 오버

### 몬스터 종류
- 🔴 **빨간 몬스터** (Red Monster)
- 🟢 **초록 몬스터** (Green Monster)
- 🟡 **노란 몬스터** (Yellow Monster)
- 🧱 **벽돌** (Brick) - 색상 변경 아이템

## 🎮 조작법

| 키 | 기능 |
|---|---|
| **A** | 슈터 왼쪽으로 이동 |
| **D** | 슈터 오른쪽으로 이동 |

## 🎨 주요 기능

### 1. 색상 매칭 시스템
- 현재 타겟 색상과 다음 타겟 색상을 시각적으로 표시
- 올바른 색상 맞추면 자동으로 다음 색상으로 전환

### 2. 생명 시스템
- 시작 시 5개의 생명
- 잘못된 색상 맞추면 생명 감소
- 생명이 0이 되면 게임 오버

### 3. 콤보 시스템 🆕
- 연속으로 올바른 색상을 맞추면 콤보 증가
- **3콤보 이상**: 1.5배 점수 (150점)
- **5콤보 이상**: 2배 점수 (200점)
- **10콤보 이상**: 3배 점수 (300점)
- 잘못된 색상을 맞추면 콤보 리셋

### 4. 하이스코어 저장 🆕
- 최고 기록 자동 저장 (highscore.txt)
- 신기록 달성 시 ★ 특별 표시
- 게임 오버 화면에 하이스코어 표시

### 5. 부기 캐릭터 감정 표현 🆕
- **생명 4개 이상**: 웃는 얼굴 (funnyBoogie)
- **생명 2-3개**: 평범한 얼굴 (defaultBoogie)
- **생명 1개 이하**: 슬픈 얼굴 (sadBoogie)

### 6. 게임 오버 처리
- 최종 점수 및 하이스코어 표시
- 신기록 달성 시 특별 애니메이션
- R키로 재시작 가능

### 7. 일시정지 기능
- P키로 언제든지 게임 일시정지 가능
- 일시정지 중에는 게임 조작 불가

### 8. 자동 타겟 재생성
- 모든 타겟을 깨면 자동으로 새로운 3줄의 타겟 생성

### 9. 멀티 스레딩
- BulletThread: 각 총알의 이동 및 충돌 처리
- MovementThread: 슈터의 부드러운 이동 처리

## 🐛 해결된 주요 버그

### 1. 벽돌 이중 깨짐 문제
- **문제**: 하나의 총알이 여러 타겟을 동시에 깨는 현상
- **해결**:
  - synchronized 블록으로 충돌 체크 동기화
  - hasHit 플래그로 중복 충돌 방지
  - 충돌 즉시 타겟을 invisible 처리

### 2. 총알 연속 통과 문제
- **문제**: 총알이 타겟을 관통하는 현상
- **해결**: 충돌 감지 시 즉시 쓰레드 종료 (return)

## 💡 게임 팁

1. **색상 미리 확인**: 다음 색상을 미리 확인하여 전략적으로 플레이
2. **벽돌 활용**: 원하는 색상이 나올 때까지 벽돌을 맞춰 색상 변경
3. **신중한 조준**: 잘못된 색상을 맞추면 생명이 감소하므로 신중하게 조준
4. **콤보 유지**: 연속으로 올바른 색상을 맞춰 높은 점수 획득
5. **일시정지 활용**: 복잡한 상황에서는 P키로 일시정지하여 전략 수립
6. **부기 표정 관찰**: 부기의 표정으로 현재 상태 파악
